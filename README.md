# Frontend Practice

## About
This repository contains my HTML and CSS practice exercises as I learn frontend development. It includes various button designs, styling exercises, typography practice, and recreations of UI elements from popular websites.

## What's Inside
- **Button Designs**: Various styled buttons with different colors, borders, and effects (Subscribe, Join, Tweet buttons)
- **Text & Typography**: Practicing font families, sizes, weights, and text styling
  - Font family exercises (Arial, Verdana, Tahoma)
  - YouTube video card styling
  - Text layout and spacing
- **CSS Exercises**: Styling practice for common UI elements
  - Amazon-style buttons and elements
  - LinkedIn-inspired designs
  - GitHub-style components
  - Bootstrap practice
  - Uber-style elements
- **HTML Basics**: Fundamental HTML practice including links, paragraphs, and page structure
- **Website Templates**: Practice website layouts and structures
